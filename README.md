# 📈 SB Stocks

A full-stack **MERN** paper trading platform that allows users to practice stock trading using **virtual funds**. SB Stocks provides real-time US stock market data, historical trends, and portfolio management features, helping users learn and test trading strategies in a risk-free environment.

## ✨ Features

* Secure User Authentication
* Real-Time US Stock Market Data
* Paper Trading with Virtual Funds
* Portfolio & Profit/Loss Tracking
* Stock Search and Historical Price Trends
* Responsive User Interface
